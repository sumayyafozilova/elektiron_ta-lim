1-dars: HTML Asoslari

HTML (HyperText Markup Language) — internetda veb-sahifalarni yaratish uchun ishlatiladigan asosiy til. HTML yordamida siz sahifalarning tuzilishini yaratishingiz mumkin.

HTML haqida batafsil

2-dars: CSS Asoslari

CSS (Cascading Style Sheets) — HTML sahifalarining ko'rinishini, ranglarini, o'lchamlarini va boshqa dizayn elementlarini boshqarish uchun ishlatiladi.

CSS haqida batafsil

3-dars: JavaScript Asoslari

JavaScript — veb-sahifalarni dinamik qilish va interaktiv funksiyalarni yaratish uchun ishlatiladi. U foydalanuvchi bilan muloqot qilish va sahifani o'zgarishlarga moslash uchun zarurdir.

JavaScript haqida batafsil

4-dars: DOM Manipulyatsiyasi

DOM (Document Object Model) — bu veb-sahifaning strukturasi. JavaScript yordamida DOM elementlari bilan ishlash va sahifani o'zgartirish mumkin.

DOM haqida batafsil

5-dars: JavaScript va CSS integratsiyasi

CSS va JavaScript-ni birgalikda ishlatish veb-sahifalarga animatsiyalar, interaktivlik va boshqa dinamik elementlarni qo'shish imkonini beradi.
